# React Fundamentals

## State

Хотя вы можете рассматривать реквизиты как аргументы, которые вы используете для настройки рендеринга компонентов, состояние похоже на хранилище личных данных компонента. Состояние полезно для обработки данных, которые изменяются с течением времени или возникают в результате взаимодействия с пользователем. Состояние дает вашим компонентам память!

Как правило, используйте реквизиты для настройки компонента при его рендеринге. Используйте состояние, чтобы отслеживать любые данные компонентов, которые, как вы ожидаете, изменятся со временем.

Следующий пример происходит в кошачьем кафе, где две голодные кошки ждут, чтобы их накормили. Их голод, который, как мы ожидаем, со временем изменится (в отличие от их имен), сохраняется как состояние. Чтобы покормить кошек, нажимайте их кнопки - это обновит их состояние.
